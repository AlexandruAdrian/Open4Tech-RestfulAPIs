Open4Tech-RestfulAPIs

This project represents a solution to the homework proposed by Open4Tech trainer Cristi Minica @ https://github.com/minicacristi/restful-apis

Info about routes

Reviews

  • GET - /reviews - Retrieves all the reviews.
  • GET - /review/:id - Retrieves one review based on the passed id
  • POST - /reviews - Requires the body to hold two JSON properties:
    • subject
    • review
  • DELETE - /reviews/:id - Deletes one review based on the passed id
  • PUT - /reviews/:id - Updates one review based on the id and requires the body to hold two JSON properties:
    • subject
    • review
  • POST - /reviews/save-to-db - Takes all the reviews saved in the memory and stores them in the database.

Weather API Proxy

  • GET - /weather - Requires a query parameter (city) and returns weather for it
